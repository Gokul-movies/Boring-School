Python Conditions and If statements



  Python supports the usual logical conditions from mathematics:


 • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b
  • Check if a string in word: a in b


These conditions can be used in several ways, most commonly in "if statements" and loops.

An "if statement" is written by using the if keyword.

conditional statement:

Example:

a = 4
b = 44
if b > a:
 print( "%s is greater than %s ",%(b,a))
elif a == b:
 print("a equal b")
else:
 print("a is greater than b")


Output:

b is greater than a

Get input

   In this example we use two variables, a and b, which are used as part of the if statement to test whether b is greater than a. As a is 4, and b is 44, we know that 44 is greater than 4, and so we print to screen that "b is greater than a".
